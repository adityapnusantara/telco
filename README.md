# Telco Customer Service AI Agent

An AI-powered customer service agent for telecommunications companies, built with FastAPI, LangChain, and Qdrant.

## Tech Stack

- **FastAPI** - Web framework
- **LangChain 1.2.11** - AI orchestration
- **OpenAI GPT-4o** - Language model
- **Qdrant Cloud** - Vector database
- **Langfuse 4.0.0** - Observability and tracing

## Installation

```bash
poetry install
```
